ConcHIP's Terms of Usage

ConcHIP (beta) software release is freely available for academic and non-commercial use.
For third party software, their original usage/licensing requirements apply.

Latest ConcHIP's bundle [Download for 64-bit (~80MB)]
The "tar" file distributed contains ConcHIP, example programs, and a brief manual for their usage. The bundle contains instructions for installing third party provers such as Redlog. The bundle also contains for your convenience executables for third party provers, Omega calculator, Mona and Z3 provers.

Latest ConcHIP's virtual machine [Download (~2.2GB)]
The "ova" VM image (Ubuntu 10.04 Desktop host) consists of ConcHIP (executable and example programs) and prerequisite software pre-installed. A brief manual for their usage can be found inside the image. The image can be imported into existing VM software such as VirtualBox (recommended) and VMWare. Although the image was tested on VirtualBox, it should also work with VMWare and others. The instructions for importing the image into VirtualBox can be found here.
